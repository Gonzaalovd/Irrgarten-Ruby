# IRRGARTEN RUBY

**Irrgarten** es un juego de laberinto desarrollado en **RUBY** en el que el jugador equipado con armas y escudos debe encontrar la salida mientras se enfrenta a monstruos que aparecen en su camino.  
El objetivo es recorrer el laberinto, sobrevivir a los combates y llegar a la salida.  
En caso de ser derrotado, el jugador se convertirá en un jugador fantasma que podrá seguir recorriendo el laberinto sin participar en combates.

## Primeros pasos

Estas instrucciones te permitirán obtener una copia del proyecto y ejecutarlo en tu entorno local para propósitos de desarrollo o prueba.

### Requisitos previos

- Tener instalado un intérprete de Ruby
- Un IDE compatible con proyectos de Ruby

### Instalación

1. Clona o descarga este repositorio.
2. Abre el proyecto en un IDE.
3. Configura el Main como entrada del programa. Este establecerá el número de jugadores, la interfaz del juego y lo iniciará con los controles.
